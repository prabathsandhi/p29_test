Technical Aspects of the Project

The frontend of this project is built using Angular, providing a dynamic single-page application (SPA). The backend is implemented using Python with the Flask framework, which handles HTTP requests and manages the authentication process via JWT (JSON Web Token). The database is MySQL, where user data and other necessary information are stored. The system is served using NGINX to host the frontend application, while the backend API runs on a separate server on port 3000. This architecture allows for a decoupled and scalable system where the frontend and backend are independently maintained.
